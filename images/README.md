# Images Folder

This folder should contain the following images for your portfolio:

## Required Images:
1. **profile.jpg** - Your professional headshot/profile picture (300x300px recommended)
2. **about.jpg** - An additional image for the about section (600x400px recommended)
3. **project1.jpg** - Screenshot or mockup of your first project (600x400px recommended)
4. **project2.jpg** - Screenshot or mockup of your second project (600x400px recommended)
5. **project3.jpg** - Screenshot or mockup of your third project (600x400px recommended)

## Image Guidelines:
- Use high-quality images (at least 1080p resolution)
- Optimize images for web (compress to reduce file size)
- Use consistent aspect ratios for project images
- Profile picture should be professional and clear
- Project screenshots should showcase your best work

## Recommended Tools for Image Optimization:
- TinyPNG (online compression)
- Photoshop/GIMP for editing
- Canva for creating mockups

## Placeholder Sources:
Until you have your own images, you can use:
- Unsplash.com for high-quality stock photos
- UI Faces for profile picture placeholders
- Dribbble or Behance for design inspiration
